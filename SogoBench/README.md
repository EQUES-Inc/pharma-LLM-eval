# SogoBench / 齟齬点検ベンチマーク

## SogoCheck

**Summary**
SogoCheck is a task focused on detecting inconsistencies within documents such as specifications and application forms. This benchmark is crucial for improving administrative efficiency in pharmaceutical operations.

**Benchmark Collection**

Notice that we do not rely on any translation of sources from other languages (e.g., English) or countries, and the benchmark comes solely from resources that are originally written in Japanese.
See our paper for more detail.


**Requirement**
vllm

**Usage**
1. Specify dataset in line 6 of `sogo_vllm.py`.
2. Specify model name or OpenAI API KEY in line 10 ~ 25 of `sogo_vllm.py`.
3. Run `python3 sogo_vllm.py`.

**Output**
- `result.csv`
