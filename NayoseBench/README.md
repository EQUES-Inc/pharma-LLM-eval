# NayoseBench / 薬学分野の名寄せベンチマーク

**Summary**
NayoseBench is a paraphrasing task aimed at standardizing drug names and substance names. This benchmark evaluates the ability to accurately identify equivalent pharmaceutical terms, addressing challenges in drug nomenclature standardization.

We release two types of dataset, one in question-answering form (NayoseQA) and the other in open-ended response form (NayoseInstruct).

## NayoseQA

Dataset is published in [Huggingface Dataset](https://huggingface.co/datasets/EQUES/Nayose-Bench-QA).

**Usage**

1. Specify the model to use in line 6 ~ 11 of `nayose-qa.py`.
2. Run `python3 nayose-qa.py`.

## NayoseInstruct

Dataset is published in [Huggingface Dataset](https://huggingface.co/datasets/EQUES/Nayose-Bench-Instruction).

Similar evaluation can be conducted using this dataset and `nayose-instruct.py`.
